# Portfolio — andrewzizzadoro.netlify.app

Andrew Zizzadoro's personal portfolio site. Lists projects with links and mini previews.

## After making changes

1. Commit with a short descriptive message
2. Push to GitHub (`git push`) — Netlify auto-deploys from the `main` branch

## Deployment

- **Live URL:** https://andrewzizzadoro.netlify.app
- **GitHub:** https://github.com/andrewzizz/portfolio

## Tech

- Single `index.html` file
- JetBrains Mono + Space Grotesk fonts
- Dark/light theme toggle
- Minimal, clean aesthetic
